# 🛡️ DFIR Investigation & Learning Lab

This repository serves as my professional portfolio and technical knowledge base, focusing on **Digital Forensics and Incident Response (DFIR)**. It hosts my personal website and detailed study notes on cyber investigation methodologies.

## 🌐 Live Website
You can visit the rendered version of this portfolio at: 
**[https://shinymate-hue.github.io/DFID/](https://shinymate-hue.github.io/DFID/)**

## 🎯 Project Goals
* **Documentation:** Consolidating DFIR artifacts, tools, and procedure notes.
* **Portfolio:** Showcasing certifications (**CCNA, PNPT, Security+**) and technical skills.
* **Continuous Learning:** Tracking progress in advanced forensics (inspired by SANS FOR508).

## 📂 Repository Structure
* `/index.md`: The main landing page for the GitHub Pages site.
* `/_config.yml`: Jekyll configuration and theme settings.
* `/notes/`: Collection of technical deep-dives into Windows Forensics, Memory Analysis, etc.

## 🛠 Tech Stack
* **Static Site Generator:** Jekyll
* **Hosting:** GitHub Pages
* **Content Format:** Markdown
